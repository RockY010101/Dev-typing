# Dev-Typing

A competitive typing application that allows users to race against each other in real-time.

## 🚀 Features

-   **Real-time Multiplayer**: Compete with other users in live typing races.
-   **Ranking System**: Track your progress with an Elo-based ranking system.
-   **Responsive UI**: Built with Tailwind CSS for a modern, responsive interface.
-   **Clean Architecture**: Modern React with custom hooks and a modular structure.

## 📋 Getting Started

### Prerequisites

Ensure you have Node.js installed (version 20 or higher is recommended).

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/RockY010101/Dev-typing.git
    cd Dev-typing
    ```

2.  Install dependencies for the client:
    ```bash
    cd client
    npm install
    ```

## 🏃‍♂️ Running the Application

### Development Mode

To run the application in development mode:

```bash
npm run dev
```

The application will start on `http://localhost:3000`.

## 🛠️ Build

To build the application for production:

```bash
npm run build
```

## 🧪 Testing

Run the test suite:

```bash
npm run test
```
