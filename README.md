## рҹ“– About

DevType is a typing practice platform built specifically for programmers.

Instead of typing random words, users practice by typing real programming syntax from different languages, helping them improve typing speed, coding familiarity, and overall accuracy.

The application combines a warm pixel-art aesthetic with focused typing sessions to create an enjoyable learning experience that feels more like coding on a rainy evening than completing another online typing test.

## рҹ“Ҡ Performance Metrics

During every session DevType calculates:

- Words Per Minute (WPM)
- Accuracy
- Total Errors
- Time Taken
- Completion Percentage

### WPM

```
WPM = (Characters Typed / 5) / Minutes
```

### Accuracy

```
Accuracy = (Correct Characters / Total Typed Characters) Г— 100
```
